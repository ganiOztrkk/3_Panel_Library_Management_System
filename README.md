
# Mvc5 İle 3 Panelli Kütüphane Yönetim Sistemi

Bir kütüphanenin vitrin, admin ve kullanıcı olmak üzere 3 temel panelini oluşturuyoruz. 
## Kullanılan başlıklar

* Layout Kontrolü
* Model Yapıları
* Linq Sorguları
* EntityFramework Sorguları
* İlişkili Tablolar
* Trigger Kullanımı
* Bootsrap
* Widget Cards
